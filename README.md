# 🛒 Carrinho de Compras

Mini projeto Java desenvolvido para praticar os conceitos de Programação Orientada a Objetos (POO).

## 📚 Conceitos praticados

- Encapsulamento (atributos privados, getters e setters)
- Construtores
- Relacionamento entre classes
- ArrayList para armazenamento dinâmico
- Sobrescrita do método `toString()`

## 🗂️ Estrutura do projeto

```
src/
├── Produto.java     # Representa um item da loja
├── Carrinho.java    # Gerencia os produtos adicionados
└── Main.java        # Ponto de entrada da aplicação
```

## ⚙️ Funcionalidades

- Adicionar produto ao carrinho
- Remover produto pelo nome
- Listar todos os produtos
- Calcular o total da compra

## ▶️ Como executar

1. Clone ou baixe o projeto
2. Abra em sua IDE (IntelliJ, Eclipse, etc.)
3. Execute a classe `Main.java`
